# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

**DO NOT** create a public issue for security vulnerabilities.

Instead:
1. Use GitHub Security Advisories (preferred)
2. Or contact @rikeda71 directly

Include:
- Description of the issue
- How to reproduce it
- Potential impact

We'll respond within 48 hours and work on a fix ASAP.

Thank you for keeping Foggo secure!
